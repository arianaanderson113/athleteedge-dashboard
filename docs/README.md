# AthleteEdge Dashboard

**Project Milestone I - Front-End Interface**  
**CIS 3300 - Software Requirements & Agile Development**  
**Georgia State University**

## Overview
Web-based dashboard for strength coaches and athletic trainers to log workouts and treatments for GSU student-athletes across all sports programs.

## Features
- Dashboard with real-time performance stats
- Athlete roster with 27 real GSU athletes
- Log workouts with exercises from actual GSU strength programs
- Log athletic training treatments and recovery protocols
- Individual athlete profiles with complete history

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Georgia State branding (Royal Blue & Red)
- Responsive design for desktop and mobile

## Running Locally

### Option 1: Direct Open
Double-click `frontend/athleteedge-gsu-final.html` to open in your browser

### Option 2: Local Server
```bash
python3 -m http.server 8000
```
Then navigate to: http://localhost:8000/frontend/athleteedge-gsu-final.html
